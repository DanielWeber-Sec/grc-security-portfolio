# Scenario: Ransomware Attack

## Initial Situation

A production system is exposed to the internet and contains a critical vulnerability.

The system is not patched in time.

---

## Attack Path

1. External attacker scans for vulnerabilities  
2. Known CVE is identified  
3. Exploitation of the service  
4. Initial access is gained  
5. Malware is deployed  

---

## Impact

- System encryption  
- Loss of availability  
- Potential data exfiltration  
- Business operations disrupted  

---

## Detection Gap

- No monitoring in place  
- No alerting  
- No anomaly detection  

→ Attack remains undetected
