# Lessons Learned

## Root Causes

- Missing patch management  
- Lack of monitoring  
- No asset visibility  
- Weak access controls  

---

## Improvements

- Implement vulnerability management process  
- Introduce SIEM / monitoring  
- Define asset inventory  
- Enforce MFA  

---

## Governance Impact

- Risk awareness increased  
- Security prioritized at management level  
- Budget for security improved  

---

## Key Takeaway

Security is not about single controls.

It is about:

- Processes  
- Visibility  
- Decision-making
